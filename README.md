# Посмотри в окно

Интерактивный каталог видео, вдохновлённый атмосферой города. Проект позволяет просматривать ролики разных населённых пунктов, фильтровать их по времени суток и находить новые места через живые кадры.

---

## Описание

«Посмотри в окно» — это клиентское приложение, которое обращается к тестовому API видео‑контента.  
Основная задача — дать пользователю возможность быстро найти и посмотреть видео из любого города, а также изучить подборки по времени съёмки: утро, день, ночь.

Проект является развитием статичной версии видео‑галереи: в этой реализации добавлены динамическая загрузка данных, фильтрация, пагинация, кастомные состояния загрузки и ошибок. Вся логика написана на **«ванильном» JavaScript** без использования фреймворков.

### Возможности

*  поиск видео по названию города (регистронезависимый);
*  множественный фильтр по времени суток;
*  бесконечная лента с кнопкой «Показать ещё»;
*  переключение основного видео кликом по карточке;
*  адаптивный прелоадер и индикация ошибок;
*  кастомный скроллбар для списка роликов.

### Используемые технологии

*  **HTML5** — семантическая разметка, шаблонизация через `<template>`;
*  **CSS3** — Grid Layout, Flexbox, кастомные свойства, стилизация скроллбара, состояния `:focus-visible`, `:hover`, `:active`;
*  **JavaScript (ES6+)** — асинхронные запросы (`fetch`, `async/await`), работа с DOM, управление событиями, модульная структура утилит;
*  **REST API** — взаимодействие с бэкендом, формирование query‑параметров для фильтрации и пагинации.

## Системные требования и запуск

Для работы проекта необходим любой современный браузер (Chrome, Firefox, Safari, Edge).  
Серверная часть не требуется — приложение полностью клиентское.

### Локальный запуск

1.  Скачайте репозиторий или склонируйте его:
    ```bash
    git clone https://github.com/Andrey-Strekalov/posmotri-v-okno-ad.git
    ```
2.  Откройте папку проекта.
3.  Запустите файл `index.html` любым удобным способом:
    *  дважды кликните по нему в файловом менеджере;
    *  используйте расширение Live Server в редакторе кода;
    *  откройте в браузере командой `open index.html` (macOS) или `start index.html` (Windows).

Никаких дополнительных зависимостей устанавливать не нужно.

## Планы по доработке

*  **Адаптивная вёрстка**  
   Сделать интерфейс удобным на мобильных устройствах и планшетах (медиазапросы, резиновая сетка).
*  **Улучшение доступности (a11y)**  
   Добавить ARIA‑метки, корректную навигацию с клавиатуры для всех интерактивных элементов, улучшить контрастность focus‑индикаторов.
*  **Сортировка контента**  
   Реализовать сортировку видео по дате добавления, названию города или продолжительности.
*  **Оптимизация производительности**  
   Внедрить ленивую загрузку изображений (lazy loading), кэширование ответов API, добавить дебаунс на ввод текста в поле поиска.
*  **Управление состоянием**  
   Вынести данные приложения в централизованное хранилище (например, на паттерне «наблюдатель») для упрощения масштабирования.
